## Installation

### Prerequistes
- Node.js 16.x

### 1. Clone

```shell
$ git clone https://github.com/minji-dev/OSS_team12.git
```

### 2. Install Packages

```bash
$ npm install
```

### 3-1. Start development server

```shell
$ npm start
```

서버 시작 후, 
[`http://localhost:3000`](http://localhost:3000) 에 접속
